# wetube

> Pages 
- Home : Show all uploaded videos.
- Login / Join / Logout / Change Password
- Search videos by keyword
- User Detail (show uploaded videos) & Edit Profile
- UpLoad Vidoe (file or record with a webcam)
- Video Detail
- Edit Video
- Comments & Reply
